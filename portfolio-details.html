DDoS Attack Detection by using Semi-Supervised Machine Learning

"DDoS Attack Detection Using Semi-Supervised Learning," focuses on identifying cyberattacks that disrupt online services. A DDoS (Distributed Denial of Service) attack occurs when hackers flood a website or server with excessive traffic, making it slow or completely inaccessible. These attacks can harm businesses, disrupt services, and cause financial losses. Traditional methods of detecting DDoS attacks rely on predefined rules and patterns, but cybercriminals continuously evolve their techniques, making these methods less effective.

To solve this problem, your project uses semi-supervised machine learning, which combines both labeled (known attack patterns) and unlabeled (unknown patterns) data. This approach allows the system to detect new and evolving DDoS attacks more accurately. Your system collects network traffic data, such as IP addresses, packet size, and transmission details, and analyzes it using machine learning algorithms like Support Vector Machine (SVM), Na√Øve Bayes, and Logistic Regression. These models classify whether the incoming traffic is normal or an attack.

Effectively detects new and evolving DDoS attacks by leveraging semi-supervised learning, allowing it to identify threats beyond predefined patterns. By reducing the need for manual labeling of attack data, it streamlines the detection process and enhances efficiency. Additionally, the system operates in real-time, providing instant alerts to administrators, enabling quick responses to potential security threats.
